# 审计日志模块

## TC-AUDIT-001: 查询审计日志列表

**测试目标**: 验证审计日志分页查询

**请求**
```bash
GET /api/v1/msp/audit-logs?page=0&size=10
Authorization: Bearer <token>
```

**Query参数**:
- `page` (可选, 默认0)
- `size` (可选, 默认10)
- `action` (可选, 如: LOGIN, CREATE_TASK, UPDATE_NODE)
- `userId` (可选)

**预期响应**
```json
{
  "success": true,
  "code": "200",
  "message": "success",
  "data": {
    "content": [
      {
        "logId": "<日志ID>",
        "userId": "<用户ID>",
        "action": "LOGIN",
        "resourceType": "AUTH",
        "resourceId": null,
        "details": "用户登录成功",
        "ipAddress": "127.0.0.1",
        "createTime": 1747891200000
      }
    ],
    "total": 1,
    "page": 0,
    "size": 10
  }
}
```
