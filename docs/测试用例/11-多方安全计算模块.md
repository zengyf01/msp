## TC-MP-001: 三方 PSI

**测试目标**: 验证三方 PSI 样本对齐（公安+税务+社保）

**前置条件**: 已启动三方节点 alice、bob、charlie

**测试步骤**:
1. 创建三方 PSI 任务
2. 指定三方参与节点
3. 配置 key_column 参数
4. 执行 PSI

**请求**
```bash
POST /api/v1/msp/tasks
Content-Type: application/json
Authorization: Bearer <token>

{
  "name": "三方PSI测试",
  "type": "PSI",
  "algorithm": "bc20",
  "participants": ["alice", "bob", "charlie"],
  "parameters": {
    "psi_type": "bc20",
    "key_column": "id_card_hash"
  },
  "description": "测试三方PSI身份核验"
}
```

**预期结果**:

- 返回匹配数量
